dependency-check:purge

Note:

This goal should be used as a Maven report.

Full name:

org.owasp:dependency-check-maven:9.0.10:purge

Description:

Maven Plugin that purges the local copy of the NVD data.

Attributes:

  • Executes as an aggregator goal.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-resources.
  • Requires that Maven runs in online mode.
